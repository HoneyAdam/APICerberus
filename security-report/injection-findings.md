## SQL Injection - SECURE

**Location:** `internal/store/*.go`

**Findings:** No exploitable SQL injection found.

**Positive patterns:**
- All queries use parameterized `?` placeholders
- ORDER BY columns validated via whitelist (`normalizeUserSortBy`)
- IN clauses built with placeholder repetition, values passed as args

**Example (audit_repo.go:185-191):**
```go
placeholders := strings.TrimSuffix(strings.Repeat("?,", len(normalized)), ",")
query := "DELETE FROM audit_logs WHERE id IN (" + placeholders + ")"
args := make([]any, 0, len(normalized))
for _, id := range normalized {
    args = append(args, id)
}
result, err := r.db.Exec(query, args...)
```

**CWE:** N/A - No vulnerability

## Path Traversal - MITIGATED

**Locations:**
- `internal/admin/server.go:475` - Config import
- `internal/mcp/config_import.go:26-28` - MCP import
- `internal/plugin/marketplace.go:134,668-671` - Plugin storage
- `internal/gateway/proxy.go:422-432` - URL path joining

**Findings:** Mitigated via multiple controls.

**Controls in place:**

1. **Config import uses os.CreateTemp** (server.go:475):
   - Path generated by Go, not user-controlled
   - Comment: "Safe: path is from os.CreateTemp, not user-controlled"

2. **MCP path argument rejected** (config_import.go:26-28):
   - SEC-GQL-010: `path` argument no longer accepted
   - Prevents arbitrary file read

3. **Plugin ID sanitization** (marketplace.go:739-742):
   ```go
   func sanitizeID(id string) string {
       re := regexp.MustCompile(`[^a-zA-Z0-9_-]+`)
       return re.ReplaceAllString(id, "-")
   }
   ```

4. **Tar path validation** (marketplace.go:668-671):
   ```go
   rel, err := filepath.Rel(pluginDir, targetPath)
   if err != nil || strings.HasPrefix(rel, "..") || rel == ".." {
       return fmt.Errorf("invalid path in archive: %s", header.Name)
   }
   ```

5. **SSRF protection** (proxy.go:301-304):
   - `validateUpstreamHost()` blocks private, loopback, link-local IPs

**CWE:** CWE-22 (Path Traversal) - MITIGATED
